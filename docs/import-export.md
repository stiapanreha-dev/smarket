# Система импорта экспорта товаров для мерчантов

### Экспорт
в кабинете мерчанта нужено сделать IU для экспорта выделенных товаров, если не выделены товары то экспортируем все товары
которые есть в кабинете мерчанта в ключая псевдоудаленные.

экспортируем в формате csv (разредитель ;)

### Импорт

Система импорта это многоступенчатая система загрузки/обновления товаров мерчанта. 
1 этап - загрузка файла (любого формата для начала csv)
Файл помещаем во временную таблицу обработки
2 этап - обработка файла
Включает ИИ анализ данных и сопоставление с нашими данными

3 этап согласование. Выводим результаты сопоставления с возможностью внести правки
т.е. на странице сопоставления выводится таблица в которой 
представлены товары из импортируемого файла и товары из нашей базы данных если это обновление
если не нашли то товары подкрашиваются как добавляемые (вставка) с возможностью поиска если вдруг система не распознала товар из нашей базы

4 этап - непосредственно импорт согласованных данных
